# Technical Portfolio: Automated Project Pipeline Monitoring and Process Optimization

## Project Overview
This repository contains a technical case study demonstrating automated business process monitoring and data architecture engineering. The project eliminates manual tracking inefficiencies by shifting data out of disconnected spreadsheets into a normalized relational reporting layer.

## Key Features & Covered Business Process Tasks
* **Project Management Support:** Implemented a target-versus-actual variance engine to flag high-risk delayed timelines.
* **Process Modeling:** Engineered a structured data relationship layer inside Power BI Desktop.
* **Data Management & Automation:** Developed an interactive corporate dashboard featuring a live KPI Donut Chart (`Count of Risk_Level`) integrated with interactive cross-functional client tables.

## Repository File Structure
* `Pipeline_Monitoring_Dashboard.pbix` - The primary interactive Power BI Desktop dashboard file.
* `Project_Operations_Backend.accdb` - The relational data scheme file.
* `tbl_Projects.xlsx` / `tbl_Milestones.xlsx` - Raw transactional operational logs (Fictional Datasets).
